Data Analyst Jobs Report

This project analyses the Australian data analyst job market using data from SEEK and Indeed, highlighting education requirements, in-demand skills, and industry trends. An interactive dashboard provides insights into market demand, guiding workforce planning and career development.

Operations & Tools: AWS (12%), Azure (6%), Git (6%), with Power BI and Tableau also in demand.
Data Packages: Pandas (16%), Spark (8%), TensorFlow (6%), with high demand for ML libraries.
